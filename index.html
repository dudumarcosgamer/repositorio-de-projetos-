Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web criada com HTML e CSS,onde aprendemos a criar um site de leitura.

A estrutura do site que utilizei em HTML, ela define a semântica e o layout do conteúdo (basicamente comandos), enquanto o CSS é usado para estilizar a página visualmente.

O HTML organiza o conteúdo com um cabeçalho, uma seção principal com livros em contêineres individuais e garante design responsivo por meio de uma meta tag viewport e layout flexível..

O CSS personaliza o visual do site com variáveis de cores, fonte do Google Fonts, esquema de cores suaves e botões interativos. A combinação de HTML e CSS permite criar um site acessível e esteticamente agradável, focando na usabilidade e na experiência do usuário, enquanto a tipografia e as cores promovem uma atmosfera acolhedora para explorar a coleção de livros.
